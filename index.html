Projects

You can access some of my project's pages:

  • Plan B Training - your own workout planner

    Client-side workout planner with 5 independent training sections (A–E). Each section features a dynamic exercise table with full CRUD functionality:

    • Add exercises with name, weight, and reps
    • Inline editing — clicking edit converts cells into editable fields
    • Delete exercises from the table
    • localStorage persistence — data is saved as JSON and survives page reload

    Built with HTML5, CSS3, and vanilla JavaScript (no frameworks).


    This project is currently in closed testing on Google Play, pending approval for release on the Play Store. The entire process of packaging and configuring it as an Android app was carried out independently.

    Want to join the testing program? Send an email to brunapessoa.bpm@gmail.com with the subject "Plan B - Tester" and you'll be added to the tester list.

  • Weatherly - Forecast you can trust

    Weather forecast SPA built with React, Vite, and Tailwind CSS, consuming the WeatherAPI and Unsplash APIs. Features implemented:

    • City search via a controlled form, supporting both click and Enter key submission
    • Geolocation on page load through the native browser API, with manual search as a fallback
    • Toggle between current weather and the 5-day forecast
    • Dynamic background — fetches a photo of the searched city via the Unsplash API
    • Loading and error states with custom messages and conditional rendering
    • Forecast list laid out with CSS Grid, aligning date, max, and min temperature columns

    React hooks used: useState, useEffect. Styled with Tailwind CSS, including translucent overlays and text-shadow for contrast over background images.

  • Tasks Management

    Task management SPA built with React 19, React Router, and Tailwind CSS. Features implemented:

    • Add tasks with title and description via a controlled form with basic validation
    • Toggle completion — visual strikethrough feedback on completed tasks
    • Delete tasks by ID
    • Details page on a separate route, with task data passed via URL query params
    • localStorage persistence using useEffect

    React hooks used: useState, useEffect, useNavigate, useSearchParams. State centralized in App.jsx with mutation functions passed down as props to child components.

  • Login Screen

    Front-end login interface with a password recovery modal. Features implemented:

    • Login form with native email validation and password field constraints (5–10 characters)
    • Submit interception via preventDefault() with redirect to a new tab
    • Password recovery modal dynamically injected into the DOM via JavaScript
    • Two-step flow in the modal — after submission, content is replaced with a confirmation message
    • Responsive design with a dedicated media queries file

    Built with HTML5, CSS3, and vanilla JavaScript.
